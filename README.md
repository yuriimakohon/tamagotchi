## Installation
1. `git clone https://github.com/yuriimakohon/tamagotchi.git`
2. Build
	 * __Unix__: `./gradlew build`
	 * __Win__: `gradlew.bat build`
3. Run
	 * __Unix__: `./gradlew run`
	 * __Win__: `gradlew.bat run`
